## 🚀 Démarrage Rapide (5 minutes)

### Étape 1 : Ouvrir le site
```
Double-cliquez sur : index.html
Voilà ! Le site s'ouvre dans votre navigateur.
```

### Étape 2 : Personnaliser vos infos
```
1. Ouvrez index.html avec un éditeur (Notepad++, VS Code, etc.)
2. Cherchez "votre.email@example.com" (Ctrl+H)
3. Remplacez par votre vrai email
4. Faites la même pour le téléphone et LinkedIn
5. Sauvegardez (Ctrl+S)
6. Rechargez le navigateur (F5)
```

### Étape 3 : Ajouter vos projets
```
1. Toujours dans index.html
2. Cherchez la section "Mes Projets"
3. Dupliquez une carte projet
4. Adaptez le titre, description, et technologies
5. Sauvegardez et rechargez
```

## 📋 Fichiers Expliqués

### 📄 index.html
**Quoi :** Le fichier principal du site
**Rôle :** Structure HTML + contenu
**À faire :** 
- ✏️ Remplacer vos infos (email, tel, LinkedIn)
- ✏️ Adapter les sections (projets, compétences, formation)
- ✏️ Ajouter des images si désiré

**Sections principales :**
```
1. Navigation (menu + logo)
2. Hero (accueil avec présentation)
3. Compétences (6 cartes technos)
4. Projets (6 projets d'apprentissage)
5. Formation (timeline avec cursus)
6. Contact (formulaire + infos)
7. Footer
```

### 🎨 css/style.css
**Quoi :** Le design visuel complet
**Rôle :** Couleurs, formes, responsive, animations
**À faire :** 
- 🎨 Changer les couleurs (optionnel)
- 📱 Valider le responsive (déjà fait ✓)
- ✨ Ajouter vos propres animations (optionnel)

**Sections clés :**
```
1. Variables de couleur (au top)
2. Composants (boutons, cartes, etc.)
3. Sections complètes (hero, projets, etc.)
4. Media queries responsive
```

## 🌐 Avant Hébergement En Ligne

### ✅ Vérifications finales
- [ ] Pas d'erreurs dans la console (F12)
- [ ] Tous les liens fonctionnent
- [ ] Responsive testé mobiel
- [ ] Formulaire teste (au moins localement)
- [ ] Pas de données perso sensibles

### 🚀 Hébergement gratuit (choisir un)

**Option 1 : Netlify** (⭐ Recommandé)
```
1. Créer repo GitHub
2. Aller sur netlify.com
3. Connecter GitHub repo
4. Auto-déploiement à chaque push
```

**Option 2 : GitHub Pages**
```
1. Créer repo "username.github.io"
2. Pousser fichiers
3. Accès automatique à https://username.github.io
```

**Option 3 : Vercel**
```
1. Aller sur vercel.com
2. Importer GitHub repo
3. Déploiement en 30 secondes
```
